Platform: PC  
Role: Lead Systems & Tools Programmer  
Game Engine: Unreal Engine 5.6  
Team Size: 8  
Designed Input Scheme: Xbox Controller  
Development Time: 12 Weeks  

## Contributions  

- Managed repo as Repository Manager utilizing Git LFS
- Created framework for gun and projectile system, allowing multiple different types of guns to be produced in a moments notice
- Created framework for hexagons, allowing varying types of hexagons to be created and functionality to be built upon it. Added debug and tools to prevent the need for individual customization
- Created artist friendly system to allow easy customization of materials on a per character basis using Unreal's Data Tables
- Created animation tooling for the team's animator to hook animations into events regarding damage types, actions and other events
- Careful documentation on debugging options and system architecture
- Optimized lights with artists to ensure consistent framerate
